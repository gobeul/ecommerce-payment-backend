Java/Spring 기반 이커머스 결제 API 백엔드 시스템
이 프로젝트는 2년차 프론트엔드 개발자의 백엔드 역량 강화를 위한 개인 프로젝트입니다. 단순한 CRUD를 넘어, 현대 이커머스 백엔드가 요구하는 인증, 결제, 성능 최적화(Redis, Kafka), 배포(Docker)까지의 전 과정을 학습하는 것을 목표로 합니다.

🚀 프로젝트 목표
Java/Spring Boot 기반의 RESTful API 설계 및 구현

Spring Security와 JWT를 이용한 안정적인 인증/인가 시스템 구축

외부 PG사 API 연동을 통한 '결제 도메인' 핵심 로직 구현

Redis, Kafka를 활용한 대용량 트래픽 대비 성능 개선 및 비동기 처리

JPA N+1 문제 해결 및 @Transactional 동작 원리 이해

Docker를 이용한 컨테이너 환경 배포 경험

💻 기술 스택 (예정)
Language: Java 17

Framework: Spring Boot 3.x, Spring Security, Spring Data JPA

Database: H2 (In-memory), MySQL (예정)

Cache: Redis

Message Queue: Kafka

DevOps: Docker
